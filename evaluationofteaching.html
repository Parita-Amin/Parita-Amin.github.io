Linked Open Dataset: Evaluation of Teaching

This is a Dataset related to the feedback that Dr. Hepting solicits from the students in the classes he teaches. It has an open license: https://creativecommons.org/licenses/by-sa/4.0/.

  • Survey Vocabulary described using Protégé: survey.owl

  • SHACL Shapes file for Data Validation: SurveyShapes

  • Comma-Separated Value (CSV) file with survey responses: FeedbackCSV

  • Sample Turtle file for Linked Data generated using OpenRefine from CSV File: CS-280-202210.ttl
