Sasktel Service Technician Questions

1. What is DNS?
Ans: Domain Name System. It maintains a records of names for IP addresses.

2. What is IP Address?
Ans: Unique number that identifies device in network.

3. What is DHCP?
Ans: Dynamic Host Configuration Protocol. It automatically assigns IP addresses to devices in network.

4. What is the difference between static and dynamic IP?
Ans: Static IP: IP address for a device in network doesn't change.
        Dynamic IP: IP address changes everytime the device connects to a network.

5. How would you check the latency of the network?
Ans: Using PING command

6. How would you check network connection on windows?
Ans: Go to Windows button, then Settings and then Network & Internet.

7. How would you check network connection on mac?
Ans: Go to Apple menu, then System Preferences and then Network.

8. What is bandwidth?
Ans: Maximum amount of data transmitted in given time.

9. Explain bandwidth to someone who doesn't understand technology.
Ans: How much data can be sent/received at a time.

10. What is the difference between bandwidth and internet speed?
Ans: Bandwidth: How much data can be transmitted.
         Internet Speed: How fast data can be transmitted.

11. What is Internet?
Ans: It is a network that connects devices worldwide.

12. Explain internet to a child.
Ans: I will explain using example of a tree. The leaves are connected to each other by branches which are part of a tree. Similarly, leaves are the devices in the network which are connected to each other by branches.
